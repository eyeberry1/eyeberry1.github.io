量子力学第一章作业

Q1

波长为λ= 1Å的X光，求光子的质量?

解:

根据德布罗意波长公式 \[ \lambda=\frac{h}{\sqrt{2Em}}\tag{1} \]

\[ \begin{align} E&=hv\\ &=h\frac{c}{\lambda}\tag{2} \end{align} \]

综合(1)式(2)式 \[ \begin{align} \lambda&=\frac{h}{\sqrt{2Em}}\\ &=\frac{h}{\sqrt{2hvm}}\\ m&=\frac{h}{2v\lambda^2}\\ &=\frac{h}{2c\lambda}\\ &=\frac{6.63\times10^{-34}}{2\times2.99792\times10^8\times1\times10^{-10}}kg\\ &=1.106\times10^{-32}kg \end{align} \]

Q2.

某金属产生光电效应的红限频率为v0 ，当用频率为v(v >v0 )的单色光照射该金属时，求从金属中逸出的光电子(质量为m)的德布罗意波长?

解:

根据红限频率为\(v_0\) \[ \text{束缚能}\ \ \ \ \ \ \ \ \ \ \ A=hv_0\tag{1} \] 由光电效应方程 \[ E_k=hv-A\tag{2} \] 根据 \[ p^2=2mE_k\tag{3} \] 德布罗意波长 \[ \lambda=\frac{h}{p}\tag{4} \] 由(1)(2)(3)(4)式可得 \[ \lambda=\sqrt{\frac{h}{2m(v-v_0)}} \]

Q3.

两个光子在一定条件下可以转化成正负电子对，如果两光子能量相等，求要实现转化的光子波长最大是多少?

解:

根据能动量关系: \[ E^2=(m_ec^2)^2+(p_ec)^2 \] 为了使转化的光子波长最大，则要求产生的电子对的动量为0

由质能方程 \[ E=hv=m_ec^2\tag{1} \] 光子的能动量关系 \[ E=pc=\frac{hc}{\lambda}\tag{2} \] 综合(1)(2)式可得 \[ \begin{align} \lambda&=\frac{h}{m_ec}\\ &=\frac{6.63\times10^{-34}J\cdot s}{0.911\times 10^{-30}Kg\times 3\times10^8m\cdot s^{-1}}\\ &=2.426\times 10^{-12}m \end{align} \]

Q4.

按玻尔理论，当电子轰击基态氢原子时，如果仅产生一条光谱线，求该电子的能量范围?

解:

根据题意,电子的能量能够满足一级到二级轨道的能量而不满足一级到三级轨道的能量

因此 \[ \begin{align} E&=T+V\\ &=-\frac{k^2me^4}{2n^2\hbar^2}\\ &=-\frac{me^4}{2\varepsilon^2h^2}\cdot \frac{1}{n^2}\\ &=E_n\ \ \ \ \ \ \ \ \ &\text{第n轨道电子总能量} \end{align} \] 电子能量范围 \[ E_2-E_1<E_e<E_3-E_1 \] 即 \[ 8.2275\times10^6J<E_e<9.59875\times10^6J \]

Q4.

射至光阴极上的光，其波长从4000Å变至3000Å，求发射出的光电子的遏制电压变化?

解: \[ \begin{align} \Delta Ue&=\frac{hc}{\lambda_1}-\frac{hc}{\lambda_2}\\ \Delta U&=\frac{hc}{e}(\frac{1}{\lambda_1}-\frac{1}{\lambda_2})\\ &=\frac{6.63\times 10^{-34}\times 3\times 10^8}{1.6\times10^{-19}}(\frac{1}{3\times10^{-7}}-\frac{1}{4\times 10^{-7}})\\ &=1.04V \end{align} \]
